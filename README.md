# Arbitrary Precision Calculator (APC)

A calculator implemented in C that supports arithmetic operations on very large numbers
beyond the limit of built-in data types.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Supports very large integers
- Implemented using linked lists

## Technologies
- C
- Data Structures (Linked List)

## How to Run
gcc *.c -o apc
./apc
